App Store

Hike Intel


Hike Intel makes planning hiking trips easier and more fun. You can share posts on hiking trails and interact with your local hiking community.
Highlight points of interest on your hike so other hikers can check it out. See areas of caution ahead of time so you can plan to avoid certain trails. Share trail conditions with other hikers so they can be better prepared to tackle the trail. Post photos of animal sightings and converse with others who have seen it too.

Technologies used: Swift, SwiftUI, CoreLocation, MapBox API, Apple Authentication, Google Sign In, Network, Push Notifications Firebase - [Analytics, Authentication, Dynamic Links, Firestore, Storage]

Recollections


Recollections makes it easy to organize your special memories in one place.
Add events to help categorize your memories. Save your memories as photos by uploading a photo from your photo library or taking a picture directly from within the app. Revisit your favorite vacations any time you want. Add captions to your photos and share puppy pictures with your friends. Flip the photo over and write notes to better remember this special memory in the future.

Technologies used: Swift, UIKit, Core Data, UserDefaults, AVFoundation, Photos

Portfolio

SJSU GO

SJSU GO serves as a modernized approach to help users interact with the GO program. Through the app, students can submit their events, see their event history and order prizes. Meanwhile, administrators are able to view and approve student submissions.
Technologies used: Swift, UIKit, Firebase

Anime Tracker

Anime Tracker connects to the Jikan API in order to query and download anime information from the Jikan database. The user's favorited animes are stored and accessed through Firebase.
Technologies used: Swift, UIKit, Firebase, Jikan API

Virtual Tourist

Virtual Tourist allows users to place pins on a map to download images of the surrounding location from the Flickr API. The user's pins and images are saved and persisted using Core Data.
Technologies used: Swift, UIKit, MapKit, Core Data, Flickr API

On The Map

On The Map lets users put pins on a shared map. The Udacity API authenticates the user and retrieves basic user information, while the Parse API populates the map with information submitted by other users.
Technologies used: Swift, UIKit, MapKit, Udacity API, Parse API

MemeMe 2.0

MemeMe 2.0 helps users create memes using images on their phones. Once an image is selected, the user can add text to the top and bottom of the image. Saved memes can be seen and edited to create new memes.
Technologies used: Swift, UIKit

Activity Roulette

Activity Roulette assists users in choosing a random activity. Once a category is selected, a query is sent to the Bored API, which returns an activity that the user can save. The user can view and delete their saved activities in the second tab.
Technologies used: Objective-C, UIKit, Bored API
